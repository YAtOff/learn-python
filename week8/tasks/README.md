## Задача 1 - max2

Напишете функция `max2`, 
която намира по-голямот измежду две числа.

Примери:

```python
>>> max2(1, 2)
2
>>> max2(3, 2)
3
```

## Задача 2 - max3

Използвайки `max2` напишете функция `max3`, 
която намира по-голямот измежду три числа.

Примери:

```python
>>> max3(1, 2, 3)
3
>>> max3(3, 2, 5)
5
```
